# 🚛 Система управления автопарком

Простой и надёжный Telegram-бот для контроля мойки автомобилей, сбора актов выполненных работ и учёта пробега.

---

## 📌 Ключевые возможности

- 📸 Водители отправляют фото мойки и актов прямо в Telegram  
- 🧾 Бухгалтер видит все акты, сгруппированные по машинам  
- 🛣 Водители раз в месяц сдают пробег, бухгалтер получает сводку  
- ⚠️ Директор получает уведомления о нарушениях графика мойки  
- 🔐 Безопасная авторизация через пригласительные ссылки  
- 🐳 Быстрый запуск одной командой благодаря Docker  

---

## 📋 Требования

- Docker и Docker Compose  
- Telegram-бот (создаётся через @BotFather)

---

## 🚀 Быстрый старт

### 1. Скопируйте проект

Скопируйте все файлы проекта в отдельную папку на вашем компьютере или сервере.

---

### 2. Создайте файл `.env`

В корне проекта есть файл `.env.example` — это шаблон.  
Создайте рядом файл `.env` и заполните его:

    SECRET_KEY=сгенерируйте-уникальный-ключ
    DEBUG=False
    DATABASE_URL=postgres://fleet_user:ваш_пароль@db:5432/fleet_bot
    DB_PASSWORD=ваш_пароль
    BOT_TOKEN=токен_вашего_бота
    CELERY_BROKER_URL=redis://redis:6379/0
    CELERY_RESULT_BACKEND=redis://redis:6379/0

Пояснения:

- SECRET_KEY — любая длинная случайная строка  
- DATABASE_URL и DB_PASSWORD — используйте одинаковый пароль  
- BOT_TOKEN — токен от Telegram-бота  
- Остальные параметры можно не менять  

---

### 3. Запустите систему

    docker compose up -d

Дождитесь появления всех контейнеров (около 1 минуты).

Проверить статус можно командой:

    docker compose ps

Все сервисы должны быть в состоянии Up.

---

### 4. Создайте учётную запись администратора (директора)

    docker compose exec web python manage.py createsuperuser

Введите имя пользователя и пароль.

---

## 🌐 Доступ

Панель администратора: http://localhost:8000/admin/  
Telegram-бот: найдите бота и нажмите /start

---

## 👥 Как работать с пользователями

### Водители

1. Отправляют боту команду /start  
2. Вводят своё ФИО  
3. Добавляют автомобиль (госномер)  

После этого водитель может:

- отправлять фото мойки  
- отправлять акты  
- сдавать пробег  

---

### Бухгалтеры и директора

Создаются через админку:

1. Зайдите в админку → Пользователи → Добавить  
2. Выберите роль Бухгалтер или Директор  
3. Скопируйте код приглашения  

Ссылка для приглашения:

    https://t.me/ваш_бот?start=КОД

Сотрудник переходит по ссылке и активирует аккаунт.

---

## 🛑 Остановка и резервное копирование

Остановить систему:

    docker compose down

Полностью удалить данные (осторожно):

    docker compose down -v

---

## 💾 Хранение данных

- postgres_data — база данных  
- media_data — загруженные фото  

Чтобы сделать резервную копию:

- сохраните Docker volumes  
- или скопируйте папку проекта  

---

## 🔧 Устранение неполадок

### «Сайт не отправил данных» при входе в админку

    docker compose exec web python manage.py migrate
    docker compose exec web python manage.py collectstatic --noinput

---

### Бот не отвечает

Проверьте токен и контейнер:

    docker compose ps bot

---

### Не приходят уведомления

Убедитесь, что пользователь нажал /start или перешёл по ссылке.
